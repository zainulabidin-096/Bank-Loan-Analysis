# Bank Loan Analysis
This project focuses on analyzing bank loan data and presenting meaningful insights through an interactive dashboard built in Microsoft Excel. The objective of this analysis is to understand loan performance, evaluate lending trends, and identify potential risk within the loan portfolio.

The dashboard provides a clear view of key financial metrics and allows users to explore the data through interactive filters. It helps in understanding how loans are performing based on repayment status, borrower profile, and loan characteristics.

Key Objectives

The main goals of this project are:

• Analyze loan application trends
• Evaluate total funded and received amounts
• Measure overall loan portfolio performance
• Identify the percentage of good and bad loans
• Understand borrower risk through financial indicators
• Create an interactive dashboard for easy exploration of loan data

Key Performance Indicators

The dashboard highlights several important KPIs that provide a quick overview of the loan portfolio.

• Total Loan Applications showing the number of loans issued
• Total Funded Amount representing the total value of loans approved by the bank
• Total Amount Received indicating the repayments collected from borrowers
• Average Interest Rate showing the average cost of borrowing
• Average Debt to Income Ratio (DTI) measuring borrower financial stability

These indicators help evaluate both the bank’s lending activity and repayment performance.

Loan Performance Analysis

The project classifies loans into two main categories to measure portfolio health.

Good Loans

Good loans represent borrowers who are paying their loans successfully.

Analysis includes:
• Total good loan applications
• Total funded amount for good loans
• Total amount received from repayments
• Percentage share of good loans within the portfolio

Bad Loans

Bad loans represent loans with higher risk such as defaults or charge offs.

Analysis includes:
• Total bad loan applications
• Total funded amount for bad loans
• Total amount received
• Percentage of risky loans within the portfolio

This comparison helps evaluate the overall risk level of the loan portfolio.

Loan Status Insights

The dashboard further analyzes loans based on their repayment status.

Loan status categories include:

• Fully Paid
• Charged Off
• Current

For each category, the dashboard analyzes:

• Number of loan applications
• Funded amount
• Amount received
• Interest rate distribution
• Debt to income ratio

This helps identify patterns in repayment behavior and potential risk areas.

Interactive Features

The dashboard includes interactive slicers that allow users to explore the data dynamically.

Users can filter the data based on:

• Loan Grade (A to G) which represents borrower credit quality
• Loan Purpose such as car loan, credit card, debt consolidation, home improvement, and others

These filters make it easier to analyze borrower segments and understand lending patterns.

Tools and Techniques Used

The following tools and techniques were used in this project:

• Microsoft Excel
• Data Cleaning and Data Preparation
• Pivot Tables
• Pivot Charts
• Dashboard Design
• Data Visualization
• Financial Data Analysis
• KPI Reporting

Key Insights

Some important insights observed from the analysis include:

• Most loans fall into the good loan category, indicating strong repayment performance
• Only a small portion of loans are classified as bad loans, representing potential credit risk
• Fully paid loans represent the largest share of completed repayments
• Loan grade and loan purpose influence borrowing patterns and repayment behavior
